# Finch Backend Java Developer Challenge

Este projeto foi desenvolvido em **Java 25** com **Spring Boot 3.5.7**, utilizando **PostgreSQL 16.x** e **Flyway** para migrações de banco de dados.  
A aplicação consome a API externa **TVMaze** para sincronizar informações de seriados e episódios.

---

## 🚀 Tecnologias
- Java 25
- Spring Boot 3.5.7
- Spring Data JPA
- Spring Security
- PostgreSQL 16.x
- Flyway
- Docker & Docker Compose
- SpringDoc OpenAPI (Swagger)
